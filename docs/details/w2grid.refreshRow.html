Refreshes one row on the screen.
refreshRow(recid, [ind])
recid string, recid of the record
ind integer, index of record in records array
Returns undefined.

Description

This method refreshes one given row on the screen. It is here to allow a quick way to refresh data really fast if only one row is changed. It will be much faster than calling .refresh, though will not refresh all changed rows, but only specified one. If ind is specified, it will use it instead of finding the record by recid.
