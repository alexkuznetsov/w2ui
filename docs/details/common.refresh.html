Refreshes the widget or a specific part of it
refresh(part)
part string, optional, specific part of the widget to refresh. If not specified, refreshes the entire widget.

Description

The part parameter is optional. If provided, it refreshes only the specified part of the widget. If not provided or null, it refreshes the entire widget. This method updates the widget's display, applies any changes, and triggers the 'refresh' event before refreshing. The method returns the time taken to refresh in milliseconds.
